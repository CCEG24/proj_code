Common, useful commands for managing or testing on the server. Replace selectors like @p or names as needed.

Command What it does
/tp <target> <destination> Teleport a player/entity to another player/entity or coordinates.
/gamemode creative|survival|adventure|spectator [player] Change gamemode for yourself or another player.
/give <player> minecraft:diamond_sword{Enchantments:[{id:sharpness,lvl:5}]} Give an item (with optional NBT like enchantments).
/time set day|night|noon|midnight Set the world time quickly.
/weather clear|rain|thunder [duration] Control the weather.
/difficulty peaceful|easy|normal|hard Change world difficulty.
/spawnpoint [player] [x] [y] [z] Set a player's respawn point.
/setworldspawn [x] [y] [z] Set the global world spawn location.
/effect give <player> minecraft:speed 60 1 true Apply a status effect (60s Speed II, hide particles).
/summon minecraft:villager ~ ~ ~ Summon an entity at your location.
/locate structure minecraft:village Find nearest structure and coordinates.
/title @a title {"text":"Welcome!","color":"green"} Show a title to players.
/scoreboard objectives add points dummy "Points" Create a simple scoreboard objective.
/scoreboard objectives setdisplay sidebar points Display the objective on the sidebar.
/scoreboard players set @p points 1 Set a player's score value.
/whitelist add <player> Add a player to the server whitelist (if enabled).
/op <player> Grant operator permissions to a player.
